# Quantum-Iris-Classifier
Quantum Machine Learning project using Qiskit to classify Iris flowers.

# Quantum Iris Flower Classification 🌸

This project demonstrates a **Quantum Machine Learning classifier** built using **Qiskit** to classify flowers from the Iris dataset.

---

## 📌 Project Overview

The goal of this project is to classify iris flowers into three categories using a **Variational Quantum Classifier (VQC)**.

Flower Classes:
- Setosa
- Versicolor
- Virginica

---

## 📊 Dataset

The project uses the **Iris Dataset**, which contains measurements of flowers.

Features used in the dataset:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## ⚙️ Technologies Used

- Python
- Qiskit
- Scikit-learn
- Matplotlib
- Google Colab

---

## 🧠 Model Used

The project implements a **Variational Quantum Classifier (VQC)**.

The model includes:

- **Feature Map** → Encodes classical data into quantum states  
- **Ansatz Circuit** → Learns patterns from the data  
- **Optimizer (COBYLA)** → Optimizes model parameters

---

## 📈 Visualization

A scatter plot is used to visualize the distribution of flower data based on sepal and petal measurements.

---

## 📊 Results

- Model Accuracy: **~66%**
- Confusion Matrix used to evaluate classification performance.
